# Project Context

This repository is a small TypeScript utility project used to experiment with an AI-assisted development workflow.

## Goals

- Practice AI-assisted coding workflows
- Keep code small, readable, and well-tested
- Use TypeScript and Vitest for simple modules

## Current Structure

src/
  greeter.ts
  format.ts

test/
  greeter.test.ts
  format.test.ts

## Development Loop

Typical workflow:

1. Propose a small feature or improvement
2. Implement the change
3. Add or update tests
4. Run `pnpm test`
5. Commit the change

## Principles

- Prefer simple, readable code
- Avoid unnecessary dependencies
- Write tests for new functionality
- Keep functions small and deterministic
